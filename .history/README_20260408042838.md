# logicfp-citation-gateway

引用网关 - 连接上层应用与本地 `logicfp-kb` 知识库的中介层

## 功能

- **知识库集成**: 统一调用本地知识库的查询、问答、对话接口
- **引用管理**: 追踪代码引用来源、版本、上下文
- **会话维护**: 管理多轮对话会话，支持上下文持久化
- **文件存储**: 以 JSON 文件形式存储所有状态（无数据库依赖）

## 快速开始

### 1. 配置环境

设置知识库环境变量：

```powershell
$env:LOGICFP_KB_HOME = "D:\workspace\python\logicfp-kb"
```

### 2. 使用示例

```python
from src.gateway import CitationGateway
from src.citation import Citation

# 创建网关
gateway = CitationGateway(kb_source="logic_fingerprint")

# 创建会话
session = gateway.create_session()

# 创建上下文
context = gateway.create_context(session.id)

# 添加引用
citation = Citation(
    source="logic_fingerprint",
    code_snippet="def example(): pass",
    description="Example function"
)
gateway.add_citation_to_context(context.id, citation)

# 加载会话
loaded_session = gateway.load_session(session.id)
```

## 核心模块说明

### CitationGateway (网关)

主要接口：
- `create_session()`: 创建会话
- `load_session(session_id)`: 加载会话
- `create_context(session_id)`: 创建上下文
- `add_citation_to_context()`: 添加引用

### 数据模型

- **Citation**: 代码引用（包含源码、来源、元数据等）
- **Context**: 引用上下文（包含多条引用、时间戳等）
- **Session**: 会话（追踪上下文集合）

### FileStorage (文件存储)

支持 CRUD 操作：
- 会话: `save_session()`, `load_session()`, `list_sessions()`, `delete_session()`
- 上下文: `save_context()`, `load_context()`, `list_contexts()`, `delete_context()`
- 引用: `save_citation()`, `load_citation()`, `list_citations(source)`
