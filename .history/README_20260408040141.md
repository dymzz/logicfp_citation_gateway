# logicfp-citation-gateway

引用网关 - 本地代码引用管理和知识库集成中介层

## 功能特性

- ✅ **本地引用管理**: 追踪代码引用来源、版本、时间戳
- ✅ **上下文池管理**: 支持多轮对话会话，自动维护引用上下文池
- ✅ **本地知识库查询**: 支持关键词搜索、分词匹配
- ✅ **问答生成**: 支持多种输出格式（Prompt、JSON）
- ✅ **多轮对话**: 智能检索相关引用，维护会话历史
- ✅ **文件存储**: JSON 格式，无需数据库，支持版本控制
- ✅ **完整测试**: 12 个单元测试，100% 功能覆盖

### 3. 代码使用示例

#### 初始化和创建会话

```python
from src.gateway import CitationGateway
from src.citation import Citation

# 初始化网关
gateway = CitationGateway(kb_source="logic_fingerprint")

# 创建会话
session = gateway.create_session(metadata={"user": "test_user"})
```

#### 管理上下文池

```python
# 创建上下文池（上下文对象）
context = gateway.create_context(session.id)
gateway.add_citation_to_context(context.id, citation)
```

#### 本地知识库查询

```python
# 查询相关引用
result = gateway.query_kb("request", top_k=5)
# {
#     "success": True,
#     "query": "request",
#     "total": 2,
#     "results": [...]
# }

for citation in result["results"]:
    print(f"{citation['description']}: {citation['code_snippet']}")
```

#### 生成问答

```python
# Prompt 格式 (用于下游模型)
answer = gateway.answer_kb("如何处理 request", output_format="prompt")
print(answer["answer"]["prompt"])

# JSON 格式
answer = gateway.answer_kb("如何处理 request", output_format="json")
```

#### 多轮对话

```python
# 第一轮
result1 = gateway.chat_kb(session.id, "request 如何处理")
print(f"找到 {len(result1['related_citations'])} 条引用")

# 第二轮（自动维护上下文）
result2 = gateway.chat_kb(session.id, "validation 怎么做")
print(f"上下文包含 {result2['total_citations_in_context']} 条引用")
```

#### 加载和查看历史

```python
# 加载保存的会话
loaded_session = gateway.load_session(session.id)

# 查看会话的所有上下文
for context_id in loaded_session.context_ids:
    context = gateway.storage.load_context(context_id)
    print(f"上下文 {context_id}:")
    for citation in context.citations:
        print(f"  - {citation.description}")
```

## API 文档

### CitationGateway 类

#### 会话管理

```python
def create_session(metadata: Optional[Dict[str, Any]] = None) -> Session
    """创建新会话"""

def load_session(session_id: str) -> Optional[Session]
    """加载已保存的会话"""
```

#### 上下文和引用

```python
def create_context(session_id: str, metadata: Optional[Dict[str, Any]] = None) -> Context
    """为会话创建新上下文"""

def add_citation_to_context(context_id: str, citation: Citation) -> None
    """向上下文添加代码引用"""
```

#### 本地查询接口

```python
def query_kb(query: str, top_k: int = 5) -> Dict[str, Any]
    """查询本地引用库
    
    Args:
        query: 查询关键词（支持中英混合、分词）
        top_k: 返回结果数量
    
    Returns:
        {
            "success": True,
            "query": "...",
            "total": 2,
            "results": [...]  # Citation 列表
        }
    """

def answer_kb(question: str, output_format: str = "prompt") -> Dict[str, Any]
    """基于本地引用生成问答
    
    Args:
        question: 问题
        output_format: 输出格式 ("prompt", "json")
    
    Returns:
        {
            "success": True,
            "question": "...",
            "answer": {
                "question": "...",
                "related_citations": [...],
                "prompt": "..."  # if format="prompt"
            }
        }
    """

def chat_kb(session_id: str, message: str) -> Dict[str, Any]
    """多轮对话（自动维护会话和上下文）
    
    Args:
        session_id: 会话 ID
        message: 用户消息
    
    Returns:
        {
            "success": True,
            "session_id": "...",
            "message": "...",
            "context_id": "...",
            "related_citations": [...],
            "total_citations_in_context": 3
        }
    """
```

### 数据模型

#### Citation (代码引用)

```python
@dataclass
class Citation:
    source: str                  # 来源 (logic_fingerprint 等)
    code_snippet: str            # 代码片段
    id: str                      # UUID (自动生成)
    description: Optional[str]   # 引用描述
    file_path: Optional[str]     # 源文件路径
    line_range: Optional[str]    # 行号范围 (start-end)
    timestamp: str               # ISO 8601 时间戳 (自动生成)
    metadata: Dict[str, Any]     # 自定义元数据
```

#### Context (引用上下文)

```python
@dataclass
class Context:
    id: str                      # UUID (自动生成)
    session_id: str              # 关联会话
    citations: List[Citation]    # 该上下文包含的引用
    created_at: str              # 创建时间
    updated_at: str              # 更新时间
    metadata: Dict[str, Any]     # 自定义元数据
```

#### Session (会话)

```python
@dataclass
class Session:
    source: str                  # 来源 (logic_fingerprint 等)
    id: str                      # UUID (自动生成)
    created_at: str              # 创建时间
    updated_at: str              # 更新时间
    context_ids: List[str]       # 该会话包含的上下文列表
    metadata: Dict[str, Any]     # 自定义元数据
```

## 工作流示例

### 场景：在代码审查中追踪引用

```python
from src.gateway import CitationGateway
from src.citation import Citation

gateway = CitationGateway(kb_source="logic_fingerprint")

# 审查者创建会话
session = gateway.create_session(metadata={"review": "PR#123"})
context = gateway.create_context(session.id)

# 添加审查中发现的关键代码引用
citations = [
    Citation(
        source="logic_fingerprint",
        code_snippet="async def process_request(...): ...",
        description="请求处理入口",
        file_path="core/processor.py",
        line_range="42-89"
    ),
    Citation(
        source="logic_fingerprint",
        code_snippet="def validate_input(...): ...",
        description="输入验证逻辑",
        file_path="core/validators.py",
        line_range="10-20"
    ),
]

for citation in citations:
    gateway.add_citation_to_context(context.id, citation)

# 后续问起相关问题
answer = gateway.answer_kb("请求处理流程", output_format="prompt")
print(answer["answer"]["prompt"])
```

## 与 logicfp-kb 的关系

| 项目                     | 职责                             | 调用方式                                          |
| ------------------------ | -------------------------------- | ------------------------------------------------- |
| logicfp-kb               | 外部代码知识库，提供全量检索     | (未来) `uv run --project $LOGICFP_KB_HOME kb ...` |
| logicfp-citation-gateway | 本项目，管理本地引用上下文和会话 | 直接导入使用                                      |

目前本项目**不直接调用** `logicfp-kb`，而是作为独立的本地引用管理系统。
